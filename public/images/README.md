# Images Directory

Place your logo file here:

- **Logo file**: `logo.png` (recommended size: 128x128px or 256x256px)
- **Path in code**: `/images/logo.png`
- **Format**: PNG with transparency (recommended) or JPG

- `public/images/logo.png` âœ… (recommended)
- `public/logo.png` (update path in Sidebar.tsx to `/logo.png`)

The Sidebar component will automatically use your logo if it exists, otherwise it will fallback to the Truck icon.
